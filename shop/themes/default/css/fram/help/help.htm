分类页面

网站首页的顶部放着超市商品的所有分类：

点击任一分类，可浏览该分类的所有商品：

分类页内容可按价格进行排序：

分类页每页显示15个商品，

页面最下方显示了该类商品的总个数及分页数：

 

搜索页面

搜索栏在页面顶部，输入关键字并点击搜索：

搜索页面内容与分类页类似：

 

商品详情

在分类或搜索页面中点击任一商品图片，

即可查看该商品详情：

 

用户登录

购买商品前用户必须先登录：

点击页面右上角的“登录”按钮进行登录。

没有账号，可点击“注册”按钮，注册帐号。

在未登录状态下进行交易，

页面都会跳转到该登录页面，帮助用户登录：

用户登录成功后，页面右上角将显示已登录的用户名：

 

购买商品

分类、搜索、商品详情页都有购买商品的按钮：

点击“购买”按钮，商品将被加入购物车，

并跳转到购物车页面：

点击加减号，可修改购买商品的数量。

也可以直接用键盘修改数量的值：

点击“删除”按钮，可将商品从购物车中删除：

购物车内的商品数量会在右上角的快捷图标中显示出来：

点击右上角的购物车图标，就能进入购物车页面：

每次点击购买的商品，都会累加在购物车中：

 

交易流程

交易流程如图，共分以下六步：

点击购物车右下角的“结算”按钮，将进入提交订单页面：

下图为订单提交页面，点击“修改”按钮可修改收货地址：

收货地址修改页面：

当用户初次交易，没有设置过收货地址时，则在提交订单前，系统会先跳转到该页面，帮助用户设置收货地址。

点击“提交订单”，订单即刻提交：

订单提交成功，点击按钮进行支付：

支付成功页面：

 

公益捐赠

在商品详情页中，有关“我要捐赠”按钮。

点击后，可在购买商品的同时，捐赠一片爱心：

商品详情中的捐赠模块，可修改捐赠小树、小苗的数量：

捐赠的树苗将与所购商品一起并入购物车中进行付款：

 

捐赠流程

捐赠页面，点击加减号修改捐赠数量：

点击“捐赠”按钮进行下一步：

点击“结算”按钮进行下一步：

点击“提交订单”按钮进行下一步：

点击按钮，进行支付：

支付成功后，捐赠完成：

 

查询捐赠记录

点击网站首页的“捐赠记录”可查询捐赠记录：

捐赠记录页面中显示着所有捐赠者的捐赠信息：
